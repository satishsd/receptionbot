WhatsApp Webhook

Incoming events from WhatsApp Cloud API

How to configure your WhatsApp webhook:
1. Go to Meta Developer Console → Your App → WhatsApp → Configuration
2. Set Callback URL to: https://your-domain.com/api/webhook/whatsapp
3. Set Verify Token to the value of WHATSAPP_VERIFY_TOKEN in your .env
4. Subscribe to the messages webhook field
5. Click Verify and Save
